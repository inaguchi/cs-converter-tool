CSファイル変換ツール

.cs を .cs.txt / .cs.md / .zip に変換できます。複数ファイル選択時はまとめて zip でダウンロードします。

ファイル投入

.cs / .txt / .md を複数読み込めます

ここにファイルをドロップ
またはクリックして複数選択
複数ファイル選択時は、どの形式を選んでも zip にまとめてダウンロードされます

プレビュー

converted.cs.txt

ここに変換後のプレビューが表示されます

GitHub Pages 公開手順

  1. GitHub で新規リポジトリを作成します。例: cs-converter-tool
  2. このHTMLを index.html という名前で保存します
  3. リポジトリのルートに index.html をアップロードしてコミットします
  4. GitHub の Settings → Pages を開きます
  5. Build and deployment の Source で Deploy from a branch を選びます
  6. Branch を main / root にして保存します
  7. 数十秒後に公開URLが表示されます
  8. 公開URLは通常 https://ユーザー名.github.io/リポジトリ名/ です
